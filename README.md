# 广东工业大学幻灯片模板
## 使用说明
1. 推荐使用overleaf；
2. 本地latex运行：下载本模板文件；
3. 编写 ```.tex``` 文件；
4. 使用 ```XeLaTeX``` 编译（当前仅支持 ```XeLaTeX``` 方式编译）。

### Overleaf使用教程：

1. 下载该源码到本地

![Alt text](image-1.png)

2. 将其导入到Overleaf(也可以先fork到自己的github中，然后从github中导入)

![Alt text](image.png)

3. 选择下载的压缩文件

![Alt text](image-2.png)
![Alt text](image-3.png)

4. 选择```XeLaTeX``` 编译即可

![Alt text](image-4.png)
